### v0.1.0
*(2015-12-21)*
* runs on Java 7
* primitive types:
  * String
  * URI
  * List_String
  * List_URI
* readers:
  * tabula format
  * comma-separated values
  * calendar
* writers:
  * tabula format
  * comma-separated values
  * HTML
  * WikiText
  * SQL
* build command:

```
$ mvn clean install
```

* release: `tabula-distribution/target/tabula-0.1.0.jar`
